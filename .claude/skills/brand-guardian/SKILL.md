# Brand Guardian

## Overview

Protect the Robinson PrintWorks identity system. Before making or reviewing brand-facing work, anchor decisions in the brand spec and keep the output consistent across products, packaging, website/app UI, photography, social, trade show, CAD/STL releases, and 3D printed objects.

Primary reference: `docs/superpowers/specs/2026-07-02-robinson-printworks-brand-identity-design.md`.

## When to Use

Use for changes involving:

- Website/app visual design, UI components, landing pages, cards, forms, icons, buttons, or motion
- Logo marks, badges, stickers, signs, printer badges, embossing/debossing, or laser engraving
- Product render prompts, photography prompts, mockup boards, packaging, social templates, or marketplace listing graphics
- Color, typography, spacing, imagery, copy tone, trust signals, or product presentation

Do not use for backend-only pricing, sync, database, or API work unless user-visible brand language changes.

## Required Brand Checks

1. Read the brand spec before deciding visual direction.
2. Preserve the existing clean engineering spirit; evolve it, do not replace it casually.
3. Default to the preferred palette: Deep Midnight Navy + restrained Electric Cyan + clean neutrals.
4. Keep cyan to precision detailing, roughly 15% of the composition.
5. Make every artifact feel like part of one Robinson PrintWorks ecosystem.
6. Treat printers, including Voron, as tools in the ecosystem, not the brand centerpiece.
7. Verify the mark/application works at small physical scale, especially the 20 mm logo requirement.
8. Prefer premium, functional, manufacturable, calm engineering aesthetics over decoration.

## Quick Reference

| Element       | Direction                                                                    |
| ------------- | ---------------------------------------------------------------------------- |
| Primary color | Deep Midnight Navy `#061B33`                                                 |
| Accent        | Electric Cyan `#22D3EE`, used sparingly                                      |
| Neutrals      | Soft White, Cool Gray, Graphite                                              |
| Mood          | Modern, engineered, premium, reliable, clean, approachable                   |

## Review Checklist

Before finishing brand-facing work, answer briefly:

- Does this match the navy/cyan precision-detailing system?
- Would it still feel Robinson PrintWorks if the logo were removed?
- Is the accent color restrained and purposeful?
- Does it look premium, engineered, and approachable?
- Does it avoid generic 3D printer hobby branding?
- Does it work across both physical printed objects and digital surfaces?

## Common Mistakes

| Mistake                                | Correction                                                                    |
| -------------------------------------- | ----------------------------------------------------------------------------- |
| Making cyan the dominant color         | Use navy/neutral as the base; reserve cyan for precision details              |
| Centering the brand on a Voron/printer | Center the ecosystem: products, tools, packaging, UI, and customer trust      |
| Adding glow/RGB/cyberpunk effects      | Use controlled studio lighting and restrained cyan highlights                 |
| Treating one asset in isolation        | Check family resemblance across product, packaging, web, and social           |
| Using generic gear/3D printer clipart  | Use refined, scalable marks and geometry that survive embossing and 20 mm use |

## Output Contract

When reviewing or proposing brand-facing work, include:

1. **Brand fit:** one sentence.
2. **Spec alignment:** bullets tied to color, form, photography, typography, or ecosystem rules.
3. **Risks:** any ways the work drifts from the system.
4. **Next adjustment:** the smallest change that improves consistency.
